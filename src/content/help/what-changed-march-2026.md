## Buyer workflow

- saved listings now support shortlist states
- compare can now produce a finalist decision
- buyers can track sent enquiries and recent views

## Owner workflow

- owner dashboards now expose listing performance signals
- owner listing drilldowns now exist per listing
- enquiry response state is now tracked

## Billing and visibility

- the billing console now includes lightweight 7-day trend reporting
- owners and admins can see more explicit feedback after listing-state changes

## Notifications and help

- saved-search freshness now feeds buyer inbox activity
- the help centre is now content-driven and linked from main navigation

## What to expect going forward

As workflows change, this updates category should explain user-visible changes in plain language rather than relying on users to infer them from the UI.
