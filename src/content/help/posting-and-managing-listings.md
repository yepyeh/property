## What makes a listing work

Owners should not think of the form as a box-ticking exercise.

A strong listing needs:

- clear pricing
- complete location context
- readable summary and description
- trust signals
- a real photo set

## Gallery expectations

The listing gallery should include:

- one exterior or building-context image
- one main living space
- one kitchen or dining image
- one bedroom
- one location or neighborhood context image

Weak or missing photos reduce trust before a buyer even reads the details.

## Managing demand

Use the owner dashboard to watch:

- views
- saves
- enquiries
- billing state

If a listing has views but no saves or enquiries, the problem is usually presentation, pricing, or trust clarity.

## When to use upgrades

- use `paid` when the listing should stay live beyond the trial
- use `promoted` when you want stronger placement for a shorter period
